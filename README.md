# Threefold-Repetition

<img width="500" alt="image" src="https://github.com/user-attachments/assets/e622912f-3825-4003-93c1-48651f9518ad" />

This is a micro printer that is heavily inspired by the voron v0 and community mods that people have created for it. The main goal of this build was to create a fast and high quality printer for an affordable price that doesn't require high voltages 48+ to generate high torques. This is because a fast printer would allow for faster prototyping, so more iterations of a products can be made quicker, thus leading to less time wasted on waiting during R&D

To achieve this I came up with a certain amount of requirements that the printer needs to meet:
- uses non physical methods for bed meshing (inductive probe / eddy current) to reduce startup time
- high flow rate nozzle
- uses 3 point maxwell couplings for perfect automatic bed leveling
- low inductance motors to reduce back emf and push acceleration higher
- small enough to fit on a desk

Cost: ~$450
